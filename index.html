Overlord

Install CLI

To install the overlord cli, you will need to use your package manager of choice from the list below.

Homebrew (Mac & Linux)

To install on Mac or Linux using homebrew use the following command `brew install overlord`

Listed below are the current supported versions available versions via homebrew

  • v1.3.0.tar.gz

Apt (Linux)

To install on Linux using apt use the following command `sudo apt install overlord`

Listed below are the current supported versions available versions via apt

  • v1.3.0.tar.gz

Windows

To install on Windows use the following command `choco install overlord`

Listed below are the current supported versions available versions via choco

  • v1.3.0.tar.gz
